电脑安装双系统(windows&ubuntu)

下载系统

  • 下载个gho文件；这里我下载的是iso镜像，然后解压提取里面的gho文件。

  • 下载系统网站

制作u盘启动

  • 用电脑店u盘工具制作的启动盘

  • 因为它有详细的视屏&文字教程

安装windows

  • 上一步制作u盘的教程中有

windows下EasyBcd引导安装ubuntu

  • 下载安装并启动EasyBcd引导软件，左侧栏EasyBcd工具箱中的添加新条目(Add New Entry) -> 操作系统(Operating System) -> Linux/BSD -> 添加条目(Add Entry)

这里的类型(type): GRUB(Legacy)；名称(Name): 可以自己取名字；驱动器(Device): 这里选择ubuntu将安装到的分区

  • 下面的便携式／外接存储设备(Portable/External Media) -> ISO(ISO Boot)

这里的路径一栏：选择下载好的iso镜像; ps: 这里需要注意，iso镜像必须放在某个分区(准备安装ubuntu的分区除外)的根目录下，并且解压一份。否则会找不到系统文件。

  • 接着重启电脑，选择“NeoSmart的ISO条目”启动项(ps:如果之前的操作步骤中<便携式／外接存储设备>没有修改名称的话)，跟着指导一步一步往下走就安装了。

这里注意：安装的时候，记得设置成windows与ubuntu共存，不要替换了。

ubuntu装好后，EasyBck设置开机引导

  • 打开EasyBcd，查看设置(View Setting) 就可以看到很多条目，这里看到的东西都会在pc启动时的启动页看到的条目。点击编辑引导菜单(Edit Boot Menu) 可以删除不需要的条目。

  • 添加开机引导项，点击添加新条目(Add New Entry) -> Linux/BSD -> 点击添加条目(Add Entry)

类型(type): GRUB(Legacy)；名称(name): 随意自己取； 驱动器(device): ubuntu安装的分区

  • 点击高级设置(Advanced Setting) -> 选择操作系统(select an os) (上一步中的name，自己取的) -> 选择Basic (drive选择ubuntu所在的驱动号) -> 保存设置

  • 重启电脑，就可以看到选项了。

  • 做到这里我的电脑已经配置好了，也许后面还有些问题，但并没有碰到。如有，可以参考这里
